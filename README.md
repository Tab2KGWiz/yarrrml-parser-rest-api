## Getting Started

### Prerequisites

Run the following command to install dependencies:

    $ npm install

### Run the project

Run the following command to run the project:

    $ npm start

## Docker

### Install

    $ docker build -t yarrrml-parser .

### Run

    $ docker run -p 3000:3000 -it yarrrml-parser

### Docker hub

    $ docker pull zihancr/yarrrml
